# Project Init

1. Collect objective, constraints, out-of-scope, acceptance checks.
2. Create/update `.agent/PROJECT-PRD.md`.
3. Ensure rules and phase plans exist.
4. Confirm validation and rollback expectations.
5. Pause for user confirmation before implementation.
