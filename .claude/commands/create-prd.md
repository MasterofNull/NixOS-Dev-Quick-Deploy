# Create PRD

Write PRD to `$ARGUMENTS` (default `.agent/PROJECT-PRD.md`) with:
1. Executive Summary
2. Mission
3. Scope (in/out)
4. Constraints
5. Architecture
6. Security & Config
7. Implementation phases
8. Validation / success criteria
9. Risks & mitigations

Ask clarifying questions if critical inputs are missing.
