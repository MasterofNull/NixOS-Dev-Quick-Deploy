# PRSI High-Risk Approval Rubric

Last updated: 2026-03-04

## Required Checklist
All items must be true before high-risk execution:
- blast_radius_assessed
- fallback_validated
- recovery_path_verified
- observability_ready
- security_impact_reviewed
- independent_verifier_signed

## Evidence Expectations
- Blast radius analysis attached.
- Recovery and rollback commands tested.
- Monitoring/alerting path verified.
- Independent verifier sign-off present.

## Record Format
Use `data/prsi-artifacts/high-risk-approval-template.json`.
