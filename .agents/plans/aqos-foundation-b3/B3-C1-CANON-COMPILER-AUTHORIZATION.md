## 3. Invariant Rules & Security Constraints

1. **Non-Authoritative Boundary:** Output generated by `aq-canon-compiler.py` is advisory/documentation only. Runtime services MUST NOT read compiler output as execution authority.
2. **Pure Determinism:** Given the same schema input, `aq-canon-compiler.py` MUST output byte-for-byte identical output files across Python environments.
3. **No Network or File System Mutation:** The compiler must operate purely in-memory, writing compiled outputs only to specified stdout or designated build target paths.
4. **Fail-Closed Validation:** Invalid schema files must cause the compiler to exit with non-zero status and print a schema validation error message.
