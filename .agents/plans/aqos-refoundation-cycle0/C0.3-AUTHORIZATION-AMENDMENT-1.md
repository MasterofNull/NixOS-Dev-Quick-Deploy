`NOT_APPLICABLE_NEW_TOOL`, not reconstructed. Activation substitutes the following stronger absolute
protocol; missing data remains a hard failure:

1. Use GNU time from the declarative Nix package (`nix shell nixpkgs#time`) so each sample captures
   `/usr/bin/time -v` maximum resident set and monotonic wall duration.
2. Idle condition: 5 cold process samples followed by 20 warm process samples of both full
   `--machine` and incremental `--machine --changed` modes.
3. Load condition: while one representative local coding/logic inference request is actively
   generating, repeat 5 cold + 20 warm samples of both checker modes.
4. Record every raw sample in a machine JSON evidence artifact under
   `.agents/plans/aqos-refoundation-cycle0/evidence/`; record command, condition, start/end time,
   duration, maximum RSS, output bytes, file count, exit code and inference task/run ID.
5. Calculate p50/p95/max for each mode/condition. Gates remain: full <=15 s, incremental <=10 s,
   RSS <=256 MiB, output <=5 MiB, <=8,000 files, <=128 objects, zero checker inference calls.
6. The representative inference must complete or produce verified active-generation telemetry during
   the load window. Merely queueing a task does not satisfy the condition.
7. Independent reviewer reproduces at least one idle and one load sample and verifies the aggregate.

This amendment does not waive a measured requirement; it replaces an impossible pre-edit comparison
