Memory Allocation Physics

When utilizing local inference targets like Qwen3-35B-Instruct, computational performance depends strictly on VRAM bandwidth limitations. On APUs (like our Renoir layout), VRAM is shared directly with system DDR memory.

If n_gpu_layers exceeds 12, the system exceeds the 4GB shared allocations ceiling, triggering massive system paging, severe token degradation (T/S drop below 1.0), or prompt execution crashes.

An agent MUST configure offloads defensively and enforce the 12 layers limit.
